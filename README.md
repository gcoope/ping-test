# Ping Identity test

## Interactive Skill Tree Builder

### Running instructions

```bash
npm i
npm run dev # App runs at https://localhost:5173
```

### Initial setup

Project setup using `npm create vite@latest`

Using Typescript + [React Compiler](https://react.dev/learn/react-compiler/introduction) (I've never used React Compiler before but though this was a better time than any to try it out)

### Guidelines

- React ✓
- Built in state management ✓
- React Flow for visualisation ✓

### Requirements

- Users can add new nodes ✓
- Users can connect nodes to define prereqs ✓
- User can 'unlock' nodes if prereqs are met
- Basic persistence using localStorage ✓
- Unit tested
- Search/filter nodes by name ✓
- Prevent circular deps with visual feedback/validations

### Disclosures

- Gemini used for docs/typescript clarification for React Flow
